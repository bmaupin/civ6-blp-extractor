Here are some resources documenting what is known so far of BLP files:

- https://civ6.fandom.com/wiki/BLP
  - Contains information on Header, Package Headers, Pointer Fixup, Type Information, Strings
- https://github.com/ghost-ng/blp-studio/wiki/BLP-101
  - This page explains what BLP files are, how they relate to SHARED_DATA and the Civilization VII game engine, and how assets flow from disk into the running game.
- https://github.com/ghost-ng/blp-studio/wiki/Resources
  - A curated collection of external references, tools, and documentation relevant to BLP Studio and Civilization VII modding.
- https://github.com/ghost-ng/blp-studio/wiki/Technical-Specs
  - Deep-dive reference for the internal structures and binary formats behind Civilization VII's asset packaging system.
- https://github.com/ghost-ng/blp-studio/wiki/BLP-Format-Specification
  - BLP Format Specification used for Civ 7 BLP files
- Civ 7 BLP files use Oodle Kraken compression
  - If Civ 6 BLP files use it as well, this tool could be useful for decompression: https://github.com/baconwaifu/ooz
